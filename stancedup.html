Stanced Up

Description

Stanced Up was my first major web project. Myself and my friend Giulio D'aliesio decided to build this website after our favourite social media app, Stanced (for sharing ideas and progress with other car enthusiasts) shut down. We wanted to create a website where anyone could show off their car build to the world. Since January 2016, we have done up over 100 features.

Things to improve

Stanced Up was my very first project, and I took it on with very minimal coding knowledge. I used a premium WordPress theme that didnt require me to code any of the functionality, and very little markup. I did customize the CSS, but for the most part, I coded very little. If I were to redo this project, I would start off with a blank canvas by using Underscores starter theme, or creating a child theme. Everything would be coded from scratch.
